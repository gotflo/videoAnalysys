Video Analysis Model details

Main Application Functionalities

  • 1. Football players, referee, and ball detection.
  • 2. Players team prediction.
  • 3. Estimation of players and ball positions on a tactical map.
  • 4. Ball Tracking.
  • 5. Distance covered calculation for each player.
  • 6. Velocity calculation for each player.

How to use?

When you start the video analysis solution, it is required to set up a match video. Additionally, we recommend configuring the settings and hyperparameters.

  • a. Upload a video to analyze, using the sidebar menu "Browse files" button.
  • b. Enter the team names that correspond to the uploaded video in the text fields in the sidebar menu.
  • c. Access the "Team colors" tab on the main page.
  • d. Select players and goalkeepers both teams color.
  • e. Follow the instructions on the page to pick each team’s colors.
  • f. Adjust hyperparameters, and select the annotation options to confohure the out frame.
  • g. Default hyperparameters are recommended
  • h. Run Analyse Video!
